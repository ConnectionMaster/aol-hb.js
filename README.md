<a name="Install"></a>

## Install

    $ clone the repository
    $ cd pub-api
    $ npm install
    
<a name="Build"></a>

## Build

    To build the project type in the terminal
    
        $ gulp build
         
    After that build results will be placed in /build directory
    
<a name="RunUnitTests"></a>    
## Run unit tests

    To run unit tests type in the terminal
    
        $ gulp tests-unit
